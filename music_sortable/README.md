# music-sortable

드래그 앤 드롭 기능을 라이브러리로 구현하여 music-player에서 사용해보려고 제작하였습니다.
<br>

## 배포하기

```bash
// 첫 배포
> npm run publish --access public

// 파일 수정 후 배포
// 이때, package.json의 버전도 같이 업그레이드 해 줘야 한다.
> npm run publish
```

## lib install-test

```bash
> npm i @thdud/sortable-list
```

## start

```bash
> npm run start
```
