# keyboard

- webpack, eslint, prettier 등 설정을 할 수 있다.
- keydown, keyup 이벤트를 통해 사용자의 키보드 입력을 감지하여 화면에 보여줄 수 있다.
- javaScript의 class를 통해 객체를 생성할 수 있다. 
<br>

## 설치하기 Installation

```bash
> npm install
```

## 시작하기 Running the app

```bash
> npm start
```

## 빌드하기 Build

```bash
> npm run build
```
